# API
/courses/
返回所有系列课

/courses/{id}/units/
返回某系列课的所有单元

/courses/{id}/lessons/
返回某系列课的所有视频课

/courses/{cpk}/units/{upk}/lessons/
返回某系列课的某单元的所有视频课

/courses/lessons/{id}/
返回某视频课的详细信息

/courses/lessons/{id}/exercises/
返回某视频课的所有练习
